ASOP (Adaptive Search for Optimal Populations) optimization algorithm

What is ASOP?

This is a Python implementation of ASOP -- Adaptive Search for Optimal Populations. ASOP is a general global optimization algorithm, that was partially inspired by ISE, Particle filter, Genetic Algorithms and other techniques and algorithms.

This is a work in progress. Unfortunately, almost no documentation is available yet. You are more than welcome to contact the author for more details, help and help suggestions.

Cloning ASOP

The best way to obtain this library is to go its Github tags page and to download the latest tag.

If you decided to clone ASOP, bare in mind that ASOP repository contains a submodule (randomArbitrary -- lirandom number generators for arbitrary distributions). Therefore, the proper way of cloning ASOP is

git clone --recursive <path-to-ASOP>

If you forgot to clone recursively, you will see an empty directory named randomArbitrary. Don't worry. Just go to the ASOP directory, and run the following command:

git submodule update --init
